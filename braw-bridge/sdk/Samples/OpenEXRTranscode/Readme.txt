A prerequisite for building this sample is OpenEXR.

OpenEXR is a multi-channel raster file format released as an open standard.
OpenEXR needs to be installed before the sample can be built and run.
Instructions for downloading and building OpenEXR and its prerequisites can be found at:

https://openexr.com

This sample was tested against OpenEXR Version 3.1

The following OpenEXR libraries are linked against:

libOpenEXR.so
libOpenEXRCore.so
libOpenEXRUtil.so

The makefile assumes the default library location for OpenEXR /usr/local/lib64
